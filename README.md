# Projector 

Command-line interface to the [.projections.json](https://github.com/tpope/vim-projectionist) format - WIP

## Installation

Add this line to your application's Gemfile:

    gem 'projector'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install projector
